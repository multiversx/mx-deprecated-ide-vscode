Inline suggestions

For the best experience when using inline suggestions, we recommend to enable the following settings in .vscode/settings.json:

  • "editor.inlineSuggest.enabled": true
  • "editor.inlineSuggest.showToolbar": true

In order to trigger inline suggestions, type Alt + \, or any other key combination that is associated with the built-in command Trigger Inline Suggestion. Then, use the Next and Previous shortcuts to navigate among the suggestions.

Environment variables

Since version v0.18.0, the extension does not automatically patch the Visual Studio Code settings anymore.

It's the responsibility of the user to adjust .vscode/settings.json and include the necessary environment variables and PATH components (for using mxpy and Rust Analyzer).
